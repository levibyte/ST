Detailed Description

This class handles all physics related actions in the engine.

Messages this subsystem listens to:

SET_GRAVITY - Sets the current gravity.

Message must contain: a pointer to a int8_t.

SET_FRICTION - Sets the current friction.

Message must contain: a pointer to a int8_t.

SET_FLOOR - Sets the floor of the level.

Message must contain: a pointer to a int32_t.

PAUSE_PHYSICS - Pauses all physics.

Message must contain: a nullptr.

UNPAUSE_PHYSICS - Resumes all physics.

Message must contain: a nullptr.
