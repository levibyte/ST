Detailed Description

This object is responsible for taking input.

Messages this subsystem listens to:

VIRTUAL_SCREEN_COORDINATES - Tell the input manager what the virtual screen resolution is.

Message must contain: a pointer to a std::tuple<int, int> containing the width and the height of the viewport.

REAL_SCREEN_COORDINATES - Tell the input manager what the actual screen resolution is.

Message must contain: a pointer to a std::tuple<int, int> containing the width and the height of the screen.

START_TEXT_INPUT - Starts taking all keyboard input and sends TEXT_STREAM messages.

Message must contain: a nullptr.

STOP_TEXT_INPUT - Stops taking all keyboard input.

Message must contain: a nullptr.

CLEAR_TEXT_STREAM - Clears the text buffer if text_input is true.

Message must contain: a nullptr.

REGISTER_KEY - Registers the given key and sends input events (pressed, held, released) for it.

Message must contain: a pointer to a ST::key.

UNREGISTER_KEY - Unregisters a previously registered key and stops sending input events for it.

Message must contain: a pointer to a ST::key.
