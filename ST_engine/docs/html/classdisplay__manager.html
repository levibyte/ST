Detailed Description

This object is responsible for managing the window.

Creates and manages an SDL_Window.

Messages this subsystem listens to:

SET_FULLSCREEN - Sets the window to fullscreen or windowed mode.

Message must contain: a pointer to a bool.

SET_WINDOW_BRIGHTNESS - Sets the window brightness. Does not currently work.

Message must contain: a pointer to a float.
