Detailed Description

This object is responsible for issuing drawing commands and drawing the current level.

Handles drawing the entities, text objects, lights, the console window and debug information on the screen.

Messages this subsystem listens to:

SET_VSYNC - Enables or disables VSYNC.

Message must contain: a pointer to a bool.

SHOW_COLLISIONS - Enables the drawing of collision boxes and coordinates.

Message must contain: a pointer to a bool.

SHOW_FPS - Show or hide the current fps.

Message must contain: a pointer to a bool.

SET_DARKNESS - Sets the current darkness level.

Message must contain: a pointer to a uint8_t.

ASSETS - Updates the internal pointer to the assets. This messages is recieved from the asset manager whenever new assets are loaded.

Message must contain: a pointer to a ST::assets* containing the assets struct.

ENABLE_LIGHTING - Enable or disable the lighting.

Message must contain: a pointer to a bool.

SET_RESOLUTION - Sets the rendering resolution for the renderer.

Message must contain: a pointer to a std::tuple<int16_t, int16_t> containing the width and the height of the screen.
