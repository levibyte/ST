Detailed Description

An allocator class that pre-allocates memory for messages.

As messages are used a lot on every single frame, it makes sense to pre-allocate them, which is what this class does. It allocates enough for 300 messages - this can be adjusted, but for the current needs of fhe engine it's enough.
