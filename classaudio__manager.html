Detailed Description

This object is responsible for playing sounds and music.

Messages this subsystem listens to:

PLAY_SOUND - Plays a sound.

Message must contain: a pointer to a std::tuple<size_t, int, int> containing the chunk to play, the volume to play it at, and the amount of times to loop it.

PLAY_MUSIC - Plays music.

Message must contain: a pointer to a std::tuple<size_t, int, int> containing the music to play, the volume to play it at, and the amount of times to loop it.

STOP_MUSIC - Stops the music.

Message must contain: a nullptr.

STOP_ALL_SOUNDS - Stops all playing sounds.

Message must contain: a nullptr.

TOGGLE_AUDIO - Toggles the audio on or off.

Message must contain: a nullptr.

ASSETS - Updates the internal pointer to the assets. This messages is recieved from the asset manager whenever new assets are loaded.

Message must contain: a pointer to a ST::assets* containing the assets struct.

SET_VOLUME - Sets the current playback volume.

Message must contain: a pointer to a uint8_t.
