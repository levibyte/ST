Detailed Description

This object represents the console window.

Displays all in-game logs and provides an interface to the lua scripting engine.

Messages this object listens to:

CONSOLE_WRITE - Writes a log to the console window.

Message must contain: a pointer to a ST::console_log object.

CONSOLE_TOGGLE - Toggles the console window.

Message must contain: a nullptr

MOUSE_SCROLL - Scrolls the console window the given amount.

Message must contain: a pointer to an int.

KEY_PRESSED - Checks if a key is pressed - only cares about the ENTER key.

Message must contain: a pointer to an ST::key object.

TEXT_STREAM - Updates the internal composition given a pointer to a std::string.

Message must contain: a pointer to a std::string.
