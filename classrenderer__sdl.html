Detailed Description

The renderer for the engine.

This renderer is based entirely on the SDL2 renderer and uses it, along with some helper methods and rendering techniques to draw textures, text, primitives to the screen.

This method will decide which of the two text rendering methods to use based on the flag that is passed, if you are unsure if your font will render properly just pass in 0 (the default for all non-internal engine text).

Parameters
argThe font to render with.
arg2The text to render.
xThe x position to render at.
yThe y position to render at.
color_fontThe color to render with.
sizeThe size of the font to render at.
flag1 to render with draw_text_cached, 0 to render with draw_text_normal and any other value will cause a method to be picked automatically.

Note that the font must previously be loaded at the selected size.
